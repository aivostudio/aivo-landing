AIVO Studio - Müzik Üret Paket (Basit Mod + Notalar)

Bu paket, AIVO Müzik Üret ekranı için hazırlanan komple içerikleri içerir:

1) UI dökümleri (Atlas mantığında notlar)
2) Basit Mod React/TypeScript komponent iskeleti
3) Eksiksiz 24'lü nota listesi (Major + Minor)
4) Gelişmiş Mod için özet notlar

Klasör yapısı:
- atlas/
- ui/
- music/keys/
- src/components/
