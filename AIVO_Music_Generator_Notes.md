# AIVO Studio – Music & Cover Generator Notes

Bu paket tamamen front‑end demo amaçlıdır.

- `studio.html`  : AIVO Studio arayüzü (Müzik Üret + Kapak Üret + Üretimlerim)
- `studio.css`   : Koyu tema, grid yerleşimi ve bileşen stil dosyası
- `studio.js`    : Mod geçişi, BPM önerileri, nota listesi, presetler ve form toplama

Gerçek üretim için, `studio.js` içindeki `console.log` ve `alert` satırları yerine
kendi backend API çağrılarınızı eklemeniz gerekir.
