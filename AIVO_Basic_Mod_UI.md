# AIVO Basic & Gelişmiş Mod – UI Alanları

## Basit Mod

- Şarkı Adı (text)
- Müzik Açıklaması (textarea, 0–200 char)
- Hızlı Öneriler (chip list – tek seçim)
- Vokal:
  - Enstrümantal
  - Erkek vokal
  - Kadın vokal
- Müzik Üret (buton)

Basit modun amacı, kullanıcıyı yormadan hızlı demo üretimi sağlamaktır.

## Gelişmiş Mod

### Üst Alanlar
- Şarkı Adı (text)
- Ruh Hali (select)
- Açıklama (textarea, 0–200 char)
- Şarkı Sözleri (textarea, opsiyonel)

### Orta Alanlar
- Tür (Genre) – select
- Tempo (BPM) – number input + öneri metni
- Karar Sesi (Key) – 24 nota (major + minor)

### Alt Alanlar
- Vokal Tipi – select
- Mix Stili – select
- Enstrüman Seçimi:
  - Türk Enstrümanları (checkbox list)
  - Modern Enstrümanlar (checkbox list)
- Ek Açıklama / Prompt – textarea

### Aksiyon
- Gelişmiş Müzik Üret – ana CTA butonu

Bu dosya, geliştiriciler için hangi alanın ne işe yaradığını anlatan kısa bir referanstır.
